## Live reload
Live reload for development configured based on [blog.logrocket.com](https://blog.logrocket.com/complete-guide-full-stack-live-reload/)

Run the nodemon development server using 

> npm run start:dev

Once run CV webpage is available at: http://localhost:4200

## HTML5 template
HTML template is taken from https://www.sitepoint.com/a-basic-html5-template/
